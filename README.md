# Bu proje, React ve Ant Design yeni öğrenenler için hazırlanmış bir başlangıç seviye görev yönetim uygulamasıdır.
Proje,React temel kavramları,Ant Design bileşenleri ve TailwindCSS kullanımını öğretmeyi amaçlamaktadır.
TaskFlow:Kullanıcıların görevlerini yönettiği tek parçadan oluşan bir web uygulamasıdır.
Web Programlama dersinde öğrendiğim React ve kendim Youtube,Yapay Zeka ve ücretsiz online platformlar aracılığıyla öğrendiğim Ant Design ve aynı şekilde bilgi edindiğim TailwindCSS ile hem öğrendiklerimi pekiştirmek hem de edindiğim bilgilerin kalıcılığını sağlamak için bu Görev Yönetim Projesini yapma kararı aldım.Umarım öğrenmeye açık arkadaşlara faydalı olmuştur.
